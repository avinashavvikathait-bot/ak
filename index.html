How to add your real project photos — GitHub Pages setup guide

Click to expand

Each image card above shows a placeholder with the exact file path you need to save your photo to. Once the images are in the right folders in your GitHub repo, they'll appear automatically. Follow these steps:

1
Create your folder structure in your repo

Inside your GitHub repository (the one that contains index.html), create an images/ folder with sub-folders for each project:

images/
├── workspace/ ← Workspace project photos
├── hospitality/ ← Kalsang Restaurant photos
├── duplex/ ← Duplex 3BHK photos
├── studio/ ← Studio Apartment photos
├── furniture/ ← Furniture Design photos
└── participation/ ← G-Town Bar photos
2
Export / screenshot pages from your portfolio PDF

Open your portfolio PDF. Take screenshots of each drawing, render or concept board — or export pages as JPG/PNG from Adobe Acrobat (File → Export → Image). Name each file exactly as shown in the grey label on each placeholder card (e.g. arch-layout.jpg). Keep filenames lowercase with hyphens.

💡 Tip: Aim for images around 1200–1800px wide. Compress them at squoosh.app before uploading to keep your site fast.
3
Upload images to GitHub

Go to your GitHub repo → navigate into the correct folder → click Add file → Upload files → drag in your images → scroll down → click Commit changes.

# Or using Git CLI:
git add images/
git commit -m "Add project portfolio images"
git push origin main
4
Replace placeholders in the HTML with real <img> tags

Find any placeholder block like this in the code and swap it for a real image tag. The path shown on the card is exactly what you put in the src attribute.

<!-- BEFORE (placeholder) -->
<div class="img-placeholder wide">...</div>

<!-- AFTER (real image) -->
<img class="gitem-img wide"
     src="images/workspace/arch-layout.jpg"
     alt="Workspace Architectural Layout"
     loading="lazy" />
💡 Click any image card to open it in the lightbox — it works for both placeholders and real images.
5
Update the lightbox image source too

Each gallery item calls openLightbox(). The lightbox pulls the image from the src attribute of the <img> automatically. You don't need to change the JS — just replace the placeholder div with your <img> tag as shown above and everything works.

# Your final repo structure should look like:
your-repo/
├── index.html
└── images/
    ├── workspace/
    │   ├── arch-layout.jpg
    │   ├── furniture-layout.jpg
    │   └── 3d-reception.jpg
    ├── hospitality/
    │   └── 3d-main-dining.jpg
    └── ...
